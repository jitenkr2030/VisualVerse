# Chemverse Sample Content (Open-Source)

This directory contains **open-source sample content** for the Chemverse platform. All content here is freely available under the MIT License.

## Contents

- **Basic Concepts**: Foundational chemistry concepts with molecular visualizations
- **Practice Problems**: Open problems and exercises for learning
- **Reference Materials**: Documentation and learning resources

## Licensing

All content in this directory is open-source and can be used, modified, and distributed freely.

## Integration

Content from this directory is loaded by `sample_loader.py` in the parent directory and integrated into the Chemverse platform as free-tier content.
