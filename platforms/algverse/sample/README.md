# Algverse Sample Content (Open-Source)

This directory contains **open-source sample content** for the Algverse platform. All content here is freely available under the MIT License.

## Contents

- **Basic Algorithms**: Foundational algorithms with step-by-step visualizations
- **Practice Problems**: Open coding challenges and exercises
- **Reference Materials**: Documentation and learning resources

## Licensing

All content in this directory is open-source and can be used, modified, and distributed freely.

## Integration

Content from this directory is loaded by `sample_loader.py` in the parent directory and integrated into the Algverse platform as free-tier content.
